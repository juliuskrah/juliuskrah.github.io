## Build and Run
To build and run locally, you need  
* [ruby][ruby]  
* [jekyll][jekyll]

Once you have jekyll installed run  
`cd juliuskrah.github.io`

```bash
bundle install
```

Jekyll

```bash
bundle exec jekyll serve
```
